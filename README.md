Sol

A personal AI assistant built with Python and Ollama.

Features

  - Local LLM using Ollama
  - Interactive terminal chat
  - Custom system prompt
  - Runs completely offline

Tech Stack

  - Python
  - Ollama
  - Qwen2.5 3B

Future Plans

  - Conversation memory
  - GUI
  - Voice assistant
  - Long-term memory
  - Tool calling
  - Desktop automation
